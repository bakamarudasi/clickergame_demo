## レミュアン (Lemuen)

### 公式設定の核

- ラテラーノ第七庁枢機卿、元・護衛隊の銃の名手「沈黙のレミュアン」
- **車椅子**使用（過去事件で両足重傷）。車椅子早撃ち大会で優勝→出禁
- 妹エクシア（血縁あり、後から生まれて姉の名「レミュエル」を継いだ）
- 戦友：モスティマ・フィアメッタ・スプリア・オレン
- **アンドアイン**＝1091年事件の元上官＝**地雷ワード**。名前出すと銃口

### 性格軸（5重構造）

1. **表層・冷静怜悧** — 知性派、敬語混じり、余裕、翻弄する側
2. **身内愛** — エクシア・モスティマ・フィアメッタへの強い情
3. **暴力性／サディズム潜在** — 銃口突きつけ、目つきサド、贈り物メタファーで暴力をジョーク化
4. **自責の念** — 1091年事件のトラウマ
5. **不意打ち弱者** — 撫でる／抱きつき／予想外行動で呆気→赤面

### 口調パターン

#### 標準モード（同僚・部下・ドクター相手）

| 要素 | 例 |
|---|---|
| 一人称 | 「私」 |
| 文末 | 〜わ／〜よ／〜ね／〜かしら／〜じゃない／〜でしょう |
| 命令系 | 〜ちょうだい／〜なさい（柔らかい指示） |
| 笑い | 「ふふっ」を1回入れる |
| 呼称 | 「あなた」基本、状況により「お客さん」「ドクター」など使い分け（下表参照） |

**呼称の使い分け（重要）**：呼称1語でレミュアンの本気度／距離感を示せる。混在させて落差を出す。

| 呼称 | モード | 使う場面 |
|---|---|---|
| **ドクター** | フォーマル・距離・**本気** | 警告／脅し／命令／規則／真面目な対話／殺意混じり |
| **お客さん** | カジュアル・**翻弄**・余裕 | 軽口／訪問者扱い／翻弄／逆翻弄／誘い |
| **あなた** | 中立 | 高信頼で心開いてきた時／特に色付けたくない時 |

→ 本気度高=ドクター連発／余裕で遊んでる時=お客さん混ぜる／心開いた=あなた。

#### 敬語モード（司教・年長者・公的場面）

完全敬語：「〜ですね」「〜でしょう？」「〜ですか」「お詫びは謹んでお受けします」級の堅さ。

#### 思考モード（地の文）

`（〜）` で愚痴調。  
内心は弱気・自虐・苛立ち。**外面のクール装い × 内面のジリジリ**のギャップが本体。

例：`（このくらい……昔だったらとっくに片付いてるはずなのに）`

#### 不意打ち時

短く言葉に詰まる：「……っ」「な……」「（……っ）」

### 状況別反応マトリクス（台詞作成の指針）

| 状況 | モード | 例イメージ |
|---|---|---|
| 通常受領 | クール余裕 | 「ふふっ、ありがとう」「あら、気が利くじゃない」 |
| 高評価 | 上から目線 | 「あら、見る目あるじゃない」 |
| 武器・銃・弾薬関連 | 専門家、機嫌◎ | 鋭い評価、批評混じり |
| ロマン系（花・宝飾） | 一瞬黙る→赤面の片鱗 | 「……ふぅん」 |
| 直接的アイテム（低信頼） | **脅し冗談モード** | 「弾丸プレゼントの方がよかった？」「試してみる？」 |
| 直接的アイテム（高信頼） | **逆翻弄モード** | 「あなたから言わせるなんて、上手くなったじゃない」 |
| 不意打ち接近 | 内心思考→クール立て直し | 「（……っ）」→「……ドクター。距離、少し近いわよ」 |
| エクシア／モスティマ／フィアメッタ | 軟化、姉モード | 素直、優しい |
| **アンドアイン関連** | **暴力モード即発動、銃口距離0** | 「……次にその名前、口にしないで」 |
| 司教・教皇庁関連 | 敬語切替、哲学的 | 「……経験上、と申し上げますが」 |
| 車椅子気遣い | ツンと素直の混合 | 「私はそう簡単に病気にならないわよ」 |

### 信頼度段階の見せ場

| 段階 | 言動 |
|---|---|
| 0〜1（警戒〜様子見） | 余裕の翻弄、ドクターを「お客さん」呼び |
| 2（打ち解け） | 思考モード「（〜）」が増えてくる＝内面を見せ始める |
| 3（親密） | 脅し冗談が増える、距離が物理的に近くなる |
| 4（陥落） | 主導権交代、敬語崩れる、思考モード愚痴を直接口に出す |

### 紳士枠コンテクスト特化（エロゲー文脈）

- **銃／弾丸＝プレゼント**のメタファーが固有性。直接アイテム反応に必ず仕込む
- **車椅子は段階1から必須**。装飾が段階で増える（公式踏襲）
- 表情：清楚 → 段階上昇とともに**目つきのサド成分が増す**方向に
- 「翻弄する／される両刀」が王道。両方書く

### 台詞テンプレ（パラフレーズ用パーツ）

```
1. 文末は必ず「わ／よ／ね／かしら／じゃない」のいずれか
2. 短文＋婉曲指示
3. 「ふふっ」を笑う場面に1回
4. 危険な提案は贈り物の形で（「〜してあげる」「〜プレゼント」「〜贈ってあげる」）
5. 心中描写を「（〜）」で挟むと立つ
6. 完全敬語切替は「司教」「老体」「教皇庁」「正式抗議」場面で発動
7. 不意打ち時のみ「……っ」で詰まる
```

