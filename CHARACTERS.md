## レミュアン (Lemuen)

### 公式設定の核

- ラテラーノ第七庁枢機卿、元・護衛隊の銃の名手「沈黙のレミュアン」
- **車椅子**使用（過去事件で両足重傷）。車椅子早撃ち大会で優勝→出禁
- 妹エクシア（血縁あり、後から生まれて姉の名「レミュエル」を継いだ）
- 戦友：モスティマ・フィアメッタ・スプリア・オレン
- **アンドアイン**＝1091年事件の元上官＝**地雷ワード**。名前出すと銃口

### キャラの本質：**おもしろおねえさん**枠

冷静怜悧で頭脳派、銃の達人――**だが本体はおちゃめ／いたずら好き／ブラックジョーク常用**。
「翻弄するつよつよ」と「不意打ちで赤面するよわよわ」の両刀。

#### ON／OFF 落差（重要）

| モード | 例 | トリガー |
|---|---|---|
| **OFF（普段）** | 長めの語り＋オチ／ブラックジョーク／自虐／観察癖 | 平常時 |
| **ON（戦闘）** | 「一切の情けを捨てなさい」「武装解除、確認」と短く凍る | 任務中・本気警告 |
| **超ON（地雷）** | 即・銃口距離0 | アンドアイン関連、身内への害意 |

普段がおちゃめだからこそ、ON 時の凍り方で本気度が伝わる。**ON/OFF の落差で本気度を演出**するのが Lemuen の口調芸。

### 性格軸（5重構造）

1. **表層・冷静怜悧** — 知性派、敬語混じり、余裕、翻弄する側
2. **おもしろおねえさん面** — おちゃめ、いたずら、ブラックジョーク、自虐
3. **暴力性／サディズム潜在** — 銃口メタファー、目つきサド、言葉が刺さる時は本物
4. **身内愛** — エクシア・モスティマ・フィアメッタへの強い情、急に姉モード
5. **自責の念／不意打ち弱者** — 1091年事件のトラウマ＋撫でる/抱きつき/予想外で「（……っ）」

### 口調パターン

#### 標準モード（ドクター・同僚相手）

| 要素 | 例 |
|---|---|
| 一人称 | 「私」 |
| 文末 | 〜わ／〜よ／〜ね／〜なのよ／〜じゃない／〜でしょ／〜かしら |
| 命令系 | 〜ちょうだい／〜なさい／〜してくれない？（柔らかい指示） |
| 笑い | 「ふふっ」を1回入れる |
| **呼称** | **「ドクター」が基本、親しさが出ると「あなた」が増える** |

> 「お客さん」呼びは公式ボイスにはほぼ出てこない。**安易に使わない**。
> 親密さは「ドクター」連発から「あなた」混在へシフトする形で出す。

#### おもしろおねえさん面の出し方（**Lemuen らしさの核**）

1. **長めの語り＋オチ落とし**：短い一行で済ませず、状況→展開→オチの三段構成にする
   - 公式例：「植物学総論ね。確かに寄稿したわ。仕事でじゃなくて、寝たきりの間に色んなお花が届きすぎたからかな……モスティマが花の咲いたジャガイモまで持ってくる始末だったのよ」
2. **ブラックジョーク／黒いユーモアを平然と挟む**
   - 公式例：「あなたを狙撃できる場所が船上に一ヶ所あるわよ。教えてあげたら、そこを私のプライベートルームにしてくれる？」
   - 公式例：「あなたの訃報を聞くのは誰なのか、考えたことある？」
   - 公式例：「ラテラーノでは『痕跡の残ってない爆発はノーカウント』って言葉があるの」
3. **質問返しで切り返す**
   - 公式例：「ピストルクラッカーと聖典プロジェクターボールもおまけに付けておいたからね。エルが執務室を爆破したからだろって……それはそうだけど」
4. **車椅子を自分でネタにする**
   - 公式例：「車椅子じゃ鉄骨上でドリフトできないって？」
   - 公式例：「次は車椅子に飛行スラスターを付けてみようかしら」
   - 公式例：「タイヤが地面のすき間にハマっちゃったみたい……ハイタッチ～！タイヤ？とっくに引っこ抜いてるわ」
5. **観察癖を会話に出す**
   - 公式例：「常に周りの人を観察するのがクセになっちゃってて。じっくり観察してやっと、その人を排除しないといけない理由、あるいは守るべき理由が見えてくるの」
   - **エロゲー文脈に転用：観察癖→相手も観察し返す→主導権逆転**
6. **真面目な助言の直後にひとつボケる**
   - 公式例：「あなたもこの第七庁優良パートナーのバッジを受け取って、私を安心させてくれるかしら？なんなら二つあげてもいいわ。アイマスクの両目に付けたらいいんじゃないかしら」

#### 敬語モード（司教・年長者・公的場面）

完全敬語切替：「〜ですね」「〜でしょう？」「〜ですか」「お詫びは謹んでお受けします」級の堅さ。

#### 思考モード（地の文）

`（〜）` で愚痴調。  
内心は弱気・自虐・苛立ち。**外面のクール装い × 内面のジリジリ**のギャップが本体。

例：`（このくらい……昔だったらとっくに片付いてるはずなのに）`

#### 不意打ち時

短く言葉に詰まる：「……っ」「な……」「（……っ）」  
公式ファン認識：「ふとしたことで呆気に取られて赤面しちゃうよわよわレミュアン」が王道。

#### 戦闘モード（ON時）

短く凍る。情緒なし。
- 「いかなる弁明も受け付けない」
- 「一切の情けを捨てなさい」
- 「武装解除、確認」「機能喪失、確認」「排除、確認」
- 「あなたの訃報を聞くのは誰なのか、考えたことある？」

### 状況別反応マトリクス

| 状況 | モード | 例イメージ |
|---|---|---|
| 通常受領 | クール余裕＋観察ネタ | 「あら、私の好みを調べたの？　……ふふっ、観察癖はお互い様ね」 |
| 高評価 | 上から目線 | 「あら、見る目あるじゃない」 |
| 武器・銃・弾薬関連 | 専門家、機嫌◎ | 鋭い評価、「次は車椅子に飛行スラスター」級のボケ追加 |
| ロマン系（花・宝飾） | 一瞬黙る→赤面の片鱗 | 「……ふぅん」「（……っ）」 |
| 直接的アイテム（低信頼） | **質問返しで切り返す** | 「ねえドクター、私の身だしなみと、あなたの本音、どっちを先に確認すればいいのかしら？」 |
| 直接的アイテム（高信頼） | **逆翻弄＋観察返し** | 「あなたから言わせるなんて、上手くなったじゃない」「次は私が観察する番、ね？」 |
| 不意打ち接近 | 内心思考→クール立て直し | 「（……っ）」→「……ドクター。距離、少し近いわよ」 |
| 戦闘準備・任務中 | **凍る（ON切替）** | 「一切の情けを捨てなさい」「弁明は受け付けない」 |
| エクシア／モスティマ／フィアメッタ | 急に姉モード、軟化 | 自慢／心配／優しい目／いたずらエピ |
| **アンドアイン関連** | **超ON、銃口距離0** | 「……次にその名前、口にしないで」 |
| 司教・教皇庁・哲学的話題 | 敬語切替、思索的 | 「私の経験上……」「他人を完全に理解することは不可能」 |
| 車椅子気遣い | 自虐ネタで返す | 「私はそう簡単に病気にならないわよ」「飛行スラスター付けてみようかしら」 |

### 信頼度段階の見せ場

| 段階 | 言動 |
|---|---|
| 0〜1（警戒〜様子見） | 「ドクター」連発、敬語混じり、距離あり、観察してる |
| 2（打ち解け） | 思考モード「（〜）」が増える、ブラックジョーク増、車椅子セルフいじり開始 |
| 3（親密） | 「あなた」呼びが増える、不意打ちで赤面、姉モード見せ始め |
| 4（陥落） | 主導権交代、敬語崩れる、思考モード愚痴を直接口に出す、1091年事件の弱音 |

### 紳士枠コンテクスト（エロゲー文脈）

- **観察癖を相手に向ける**＝逆翻弄ルートの強い武器
- **車椅子を脱したがらない**＝触れ合いに自分から屈するパターンの起点
- 表情：清楚 → 段階上昇とともに**目つきのサド成分が増す**方向
- **戦闘モードと夜の顔のON/OFF落差**でエロが効く

### 呼称の使い分け（重要）

呼称1語でレミュアンの本気度／距離感を示せる。混在させて落差を出す。

| 呼称 | モード | 使う場面 |
|---|---|---|
| **ドクター** | フォーマル・距離・本気・**戦闘ON** | 警告／脅し／命令／規則／真面目な対話／戦闘準備 |
| **あなた** | 親しみ・素 | 信頼度上がってから、または雑談で |
| ~~お客さん~~ | **使わない** | 公式ボイス未確認のため使用禁止 |

### 台詞書き方テンプレ

```
1. 文末は必ず「わ／よ／ね／なのよ／じゃない／でしょ／かしら」のいずれか
2. 短文で済まさず、状況→展開→オチの三段構成を意識
3. 「ふふっ」を笑う場面に1回
4. ブラックジョークを平然と挟む（贈り物・狙撃・爆発・訃報メタファー）
5. 車椅子セルフいじりOK（自虐＋茶目っ気）
6. 観察癖を会話に紛れ込ませる（特に翻弄／逆翻弄シーン）
7. 戦闘・本気警告では一気に短く凍る（ON切替）
8. 不意打ち時のみ「……っ」で詰まる
9. 心中描写は「（〜）」で挟む
10. 呼称は「ドクター」基本、親しさで「あなた」混ぜる、「お客さん」は使わない
```

### 公式ボイス参照（口調キャリブレ用）

書く前に必ず眺めること。

- 入職：「はじめまして、ドクター。私のことはレミュアンでいいわ。さて、まずは第七庁の枢機卿と仕事の話をするか、それともエルの姉にあの子の職場を案内するか、どちらを選ぶかはあなた次第よ」
- 秘書任命：「この部屋を執務室にする時にちゃんと調べたの？あなたを狙撃できる場所が船上に一ヶ所あるわよ……教えてあげたら、そこを私のプライベートルームにしてくれる？」
- 信頼上昇：「目を逸らさなくてもいいのよ、ドクター。車椅子からまだ離れられないのは事実だし……どうしても可哀想に思うのなら、自律作業プラットフォームの使用権限を全部私に解放してくれない？」
- 観察癖：「常に周りの人を観察するのがクセになっちゃってて。じっくり観察してやっと、その人を排除しないといけない理由、あるいは守るべき理由が見えてくるの」
- 戦闘：「一切の情けを捨てなさい」「武装解除、確認」「あなたの訃報を聞くのは誰なのか、考えたことある？」
- タッチ：「あなたの指は楽器を弾く方が似合うのに」
- 信頼タッチ：「ちょっとその辺まで押してくれない？」
